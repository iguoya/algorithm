# 分治思想

# 算法思想

将一个复杂问题分成两个或者多个相似的子问题，子问题继续划分，直到子问题可以简单求解。

原问题的解是子问题的合并

# 案例

归并排序，二叉树遍历，傅里叶变换

# 具体策略

最常见的分治法是二分法。 三分法？？？

每次将问题难度降低为原问题的一半

# 基本步骤

  1. 分解，递归分层生成子问题
  2. 解决子问题，返回子结果
  3. 逐层合并子结果
